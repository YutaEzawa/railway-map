# 設計書: 千葉県 鉄道路線図

国土地理院の**白地図タイル**を背景に、千葉県内の鉄道（JR 東日本＋私鉄・三セク）の全路線・全駅を**実緯度経度**で表示する Web アプリの設計書。

## 概要

- **目的**: 千葉県の鉄道路線を、地理的に正確な位置で、運行本数の多寡を線の太さで表現して可視化する趣味アプリ。
- **特徴**:
  - 白地図（国土地理院 `xyz/blank`）＋ 実緯度経度の路線・駅。
  - 路線データは国土数値情報 鉄道データ N02 から生成（クロール不使用）。
  - 凡例のチェックボックスで **JR / 私鉄** の表示切替。
  - 運行本数（平日・片方向）で線種を **太線・普通・細線・点線**の 4 段階。
  - 総武線・常磐線の**複々線は各駅停車／快速を並べて表示**。
  - 駅名ラベルは**ズームに応じて段階表示**（広域は主要駅のみ）。
  - 路線・駅を**クリックすると情報ポップアップ**。
- **ホスティング**: Cloudflare Pages（`main` への push で自動デプロイ）。従量課金サービスは使わない。
- **出典**: 国土地理院（白地図）、国土数値情報（鉄道データ）。地図 attribution に表示。
